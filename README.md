# Food Delivery Web Application (QuickBite)

A full-stack food delivery web application where users can browse food items, place orders, make payments and track their orders in real time.

## Technologies Used
- **Backend:** Java, Spring Boot
- **Database:** MySQL, Spring Data JPA
- **Frontend:** HTML, CSS, JavaScript
- **Build Tool:** Maven

## Features
- User Registration and Login
- Browse Food Menu
- Add to Cart and Place Orders
- Payment Page
- Order Tracking
